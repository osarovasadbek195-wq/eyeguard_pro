# Changelog

All notable changes to the EyeGuard Pro project will be documented in this file.

## [1.0.0] - 2026-05-03

### Added
- Initial release of EyeGuard Pro
- Onboarding flow with 3 steps (filter test, camera test, break test)
- Dashboard with feature toggles
- Blink Reminder (simple and advanced modes)
- Break Reminder with 20-20-20 rule
- Distance Monitor with AI face detection
- Blue Light Filter with overlay
- Eye Exercises with gamified tracking
- Statistics with fl_chart graphs
- Multi-profile support
- Children Mode with PIN protection
- Autostart functionality (Android & Windows)
- HyperOS 3 premium theme with glassmorphism
- Dark mode support
- Offline-first architecture (no internet required)
- Local storage with Hive and Isar

### Technical
- Flutter 3.24+ with Riverpod state management
- Google ML Kit for face detection
- Platform channels for native features
- Background services for timers and notifications
